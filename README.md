## 🤖 Agentic Development: Cursor & AI Orquestração

> **Status do Projeto:** Concluído ✅  
> **Nível:** Avançado / DIO Challenge

---

## 📖 1. Visão Geral
Este projeto documenta o uso estratégico do editor **Cursor** como uma central de Inteligência Artificial. O objetivo é transformar a IA de um simples chat em um **Agente Integrado** que compreende toda a base de código (*codebase*), acelerando o desenvolvimento e garantindo a qualidade técnica.

---

## ⚙️ 2. Ecossistema de Modelos (LLMs)
O ambiente foi configurado para alternar entre os melhores motores de IA do mercado, garantindo versatilidade:

* **GPT-4o & Claude 3.5 Haiku:** Raciocínio lógico e refatoração profunda.
* **Ollama:** Integração para execução de modelos locais.
* **GPT-5 mini (Preview):** Exploração de tecnologias de vanguarda.

---

## 🛠️ 3. Modos de Interação: O Diferencial Estratégico

O domínio das ferramentas foi dividido em três pilares fundamentais de autonomia:

| 🔹 Modo | 🎯 Função | 🚀 Aplicação Prática |
| :--- | :--- | :--- |
| **Ask** | Mentor Técnico | Explicação de bugs, leitura de logs e teoria sem mudar o código. |
| **Agent** | Dev Autônomo | Criação de arquivos, automação de features e execução de comandos. |
| **Plan** | Estrategista | Mapeamento lógico e arquitetura antes de iniciar o código. |

---

## 🧠 4. Engenharia de Contexto
Diferente de IAs comuns, o **Agente no Cursor** possui consciência total do projeto:
* **Contexto Integrado:** Entendimento das relações entre diferentes pastas e arquivos.
* **Execução Direta:** No modo *Agent*, a IA executa mudanças em múltiplos componentes simultaneamente para entregar uma funcionalidade completa.

---

## 📂 5. Repositório de Prompts (Boas Práticas)
Para manter o padrão **Clean Code**, estruturamos prompts versionados:
1.  `prompt-agent.txt`: Instruções para execução e automação.
2.  `prompt-ask.txt`: Consultoria e revisão de código.
3.  `prompt-study.txt`: Guia de aprendizado e documentação.
